# Ionic 4 App

Created with:

```
ionic start ionic4-app blank --type=angular --capacitor
cd ionic4-app
npx cap add ios
npx cap add android
```

## Building

```
npm ci
ionic build
```

## Running Serve

```
ionic serve
```

### Running iOS

```
npx cap open ios
```

Build and run with Xcode.

### Running Android

```
npx cap open android
```

Build and run with Android Studio.
